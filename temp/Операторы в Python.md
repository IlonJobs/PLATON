### 1. Арифметические операторы

С ними ты знаком еще со школы, но в Python есть пара «фишек», которые экономят кучу времени.

|**Оператор**|**Описание**|**Пример**|
|---|---|---|
|`+`|Сложение|`5 + 2 = 7`|
|`-`|Вычитание|`5 - 2 = 3`|
|`*`|Умножение|`5 * 2 = 10`|
|`/`|Деление (всегда дает дробь)|`5 / 2 = 2.5`|
|`//`|**Целочисленное деление** (отбрасывает остаток)|`5 // 2 = 2`|
|`%`|**Остаток от деления**|`5 % 2 = 1`|
|`**`|**Возведение в степень**|`5 ** 2 = 25`|

> **Кстати:** Оператор `%` — это лучший друг программиста, когда нужно проверить, является ли число четным (`x % 2 == 0`).

---

### 2. Операторы сравнения

Они всегда возвращают логическое значение: либо `True` (Истина), либо `False` (Ложь).

- `==` — Равно (не путай с одинарным `=`, который присваивает значение!)
    
- `!=` — Не равно
    
- `>` / `<` — Больше / Меньше
    
- `>=` / `<=` — Больше или равно / Меньше или равно
    

---

### 3. Логические операторы (Boolean Operators)

Нужны для создания сложных условий. В Python они пишутся словами, что делает код похожим на английский язык:

1. **`and`** (И): Возвращает `True`, если **оба** условия верны.
    
2. **`or`** (ИЛИ): Возвращает `True`, если хотя бы **одно** условие верно.
    
3. **`not`** (НЕ): Инвертирует результат (делает из правды ложь и наоборот).
    

---

### 4. Операторы присваивания

Помимо обычного `=`, существуют комбинированные операторы. Они позволяют изменять переменную «на месте», делая код чище.

- `x += 5` — то же самое, что `x = x + 5`
    
- `x -= 3` — то же самое, что `x = x - 3`
    
- `x *= 2` — то же самое, что `x = x * 2`
    

---

### 5. Операторы принадлежности (Identity & Membership)

Это «изюминка» Python, которая делает его очень выразительным.

- **`in`**: Проверяет, находится ли элемент внутри последовательности (например, есть ли буква в строке или число в списке).
    
    - _Пример:_ `'a' in 'apple'` → `True`
        
- **`is`**: Проверяет, являются ли два объекта одним и тем же объектом в памяти (не просто равны по значению, а идентичны).
    

---

### Важный нюанс: Приоритет операций

Python соблюдает математические правила: сначала возведение в степень, потом умножение/деление, и только в конце сложение/вычитание. Если хочешь изменить порядок — используй **скобки `()`**.

$$x = (2 + 3) * 4$$

Здесь сначала выполнится сложение, и результат будет `20`, а не `14`.
